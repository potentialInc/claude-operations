## 인증/로그인

### 세션 만료/자격증명 오류 반복 노출 (빈도: 5회)
- **예방 스펙**: 세션 만료(401/403) 시 자동 토큰 갱신, 갱신 실패 시 로그인 리다이렉트, 에러 1회만 표시, 전역 에러 상태 관리

### 로그인/회원가입 동작 불가 (빈도: 5회)
- **예방 스펙**: 로그인/회원가입 버튼은 유효성 검사와 무관하게 클릭 가능, 검사 실패 시 구체적 오류 표시. 로그아웃 API는 토큰 만료와 무관하게 성공 반환

### 인증 후 잘못된 화면으로 리다이렉트 (빈도: 4회)
- **예방 스펙**: 모든 라우트에 인증 가드 적용. 미인증 → 로그인. 인증됨 → 로그인페이지 접근 시 대시보드로. 가입 완료 후 네비게이션 스택 초기화

### 비밀번호 유효성 검사 미흡 (빈도: 3회)
- **예방 스펙**: 비밀번호 실시간 유효성 검사, 규칙 미충족 시 구체적 안내. 변경 API는 명확한 성공/실패 응답

## 이미지/파일 업로드

### 파일 업로드 실패 및 요구사항 오류 (빈도: 3회)
- **예방 스펙**: 업로드 전 허용 파일 형식/최대 크기 안내, 미충족 시 구체적 에러. 서버도 MIME 타입 화이트리스트 + 최대 크기 검증

### 프로필 이미지 변경 실패 (빈도: 3회)
- **예방 스펙**: 업로드 전 최대 2MB 압축, 업로드 성공 후 캐시 무효화. 서버 응답에 새 이미지 URL 포함

---

## 리스트/페이지네이션

### 페이지네이션 미구현 및 정렬 오류 (빈도: 6회)
- **예방 스펙**: 목록 API는 page/page_size 파라미터 지원, 기본 정렬 created_at DESC. 10개+ 예상 목록에 페이지네이션/무한스크롤 + empty state

### 필터 적용 후 자동 초기화 (빈도: 5회)
- **예방 스펙**: 필터 조건은 상위 컴포넌트/전역 상태 보관, 모달 재오픈 시 이전 선택값 유지. 필터 초기화 버튼 별도 제공

### 뒤로가기 시 페이지/검색 상태 초기화 (빈도: 2회)
- **예방 스펙**: 페이지 번호/검색어/필터를 URL query parameter 또는 전역 상태에 저장

## 실시간 데이터

### 실시간 메시지/알림 미수신 (빈도: 5회)
- **예방 스펙**: WebSocket 연결을 앱 레벨 전역 관리. 포그라운드 복귀 시 연결 확인/재연결. 상태 변경 시 실시간 이벤트 emit. 소켓 연결 해제 시 놓친 이벤트 재조회

---

## 관리자 대시보드

### 어드민 필터/통계 오류 (빈도: 6회)
- **예방 스펙**: '전체' 필터 시 전체 데이터 반환. 통계 API는 모든 상태값 집계. 통계 데이터는 화면 포커스 시 재조회 또는 30초 자동 갱신

### 관리자 벌크/단일 액션 오류 (빈도: 3회)
- **예방 스펙**: 어드민 API에 RBAC 적용, 모든 상태 전환 API 사전 구현/테스트

---

## 검색/필터

### 검색 기능 미동작 (빈도: 6회)
- **예방 스펙**: 검색 입력에 300ms 디바운스, API 쿼리 파라미터로 전달. 0건 시 empty state. 서버는 ILIKE 부분 일치 지원

---

## 네비게이션/라우팅

### 뒤로가기 버튼 잘못된 리다이렉션 (빈도: 3회)
- **예방 스펙**: navigate(-1) 대신 명시적 목적지 경로 사용. 페이지별 기본 뒤로가기 목적지 스펙 명시

### 페이지 리로드 시 첫 페이지로 리다이렉트 (빈도: 1회)
- **예방 스펙**: 보호된 라우트 평가는 인증 상태 복원 완료 후 실행. 초기 로딩 중 스피너 표시

---

## 폼/입력

### 폼 유효성 검사 미동작 (빈도: 7회)
- **예방 스펙**: 모든 입력 필드 실시간 유효성 검사. Submit 버튼은 모든 필수 필드 유효 시에만 활성화. 각 필드 유효성 상태 개별 추적

---

## UI/레이아웃

### UI 컴포넌트 불일치 (빈도: 8회)
- **예방 스펙**: 공통 UI는 디자인 시스템 컴포넌트 재사용. 이미지/데이터 없음 상태 placeholder 필수. 프로필 정보는 전역 상태 관리

### i18n 키 누락 (빈도: 1회)
- **예방 스펙**: i18n 키 추가 시 ko.json, en.json 동시 등록. CI에서 누락 키 감지 린트 적용

---

## API/서버

### API 500 에러 (빈도: 2회)
- **예방 스펙**: UPDATE 엔드포인트는 partial update 지원, null 필드 무시. 전역 예외 핸들러로 미처리 예외 500 방지
