## 인프라
| 항목 | 클라이언트 언급 |
|------|--------------|
| 실시간 | "실시간 반영되면 좋음" — WebSocket 또는 SSE 필요 |
| 멀티테넌시 | "조직별 데이터 완전 분리" — 테넌트 격리 필수 |
| i18n | "EN/KO" — 2개 언어 지원 |
| 반응형 | "반응형 웹" — 모바일/태블릿/데스크톱 |

## 특이사항
- 기술 스택 명시 없음 → 에이전트가 best practice 기반 권장안 작성
- 실시간 기능 요구 → WebSocket 고려 필요
- 이미지 위 핀 좌표 → Canvas 또는 절대좌표 기반 오버레이 필요
- 버전 비교 (overlay/side-by-side) → 이미지 비교 라이브러리 필요
- 향후 디자인 작업 리뷰 확장 → 플러그인/어댑터 패턴 고려
