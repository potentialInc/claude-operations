# Tech Hints — 타이어뱅크 자사몰

## 기술 선호
| 항목 | 클라이언트 언급 | 출처 |
|------|--------------|------|
| Frontend Framework | React | 입력파일 기술스택 |
| Backend Framework | NestJS | 입력파일 기술스택 |
| Database | PostgreSQL | 입력파일 기술스택 |
| 지도 API | 카카오맵 API | 입력파일 기술스택 |
| 결제 | PG 연동 | 입력파일 기술스택 |
| 알림 | SMS / 알림톡 연동 | 입력파일 기술스택 |
| 차량 조회 | 외부 연동 | 입력파일 기술스택 |

## 3rd Party 서비스
| 서비스 | 용도 | 클라이언트 언급 |
|--------|------|--------------|
| 카카오맵 API | 장착점 지도 표시/검색 | 명시적 언급 |
| 차량조회 API | 차량번호 기반 차량 정보 조회 | "외부 연동"으로 언급 |
| PG (결제 게이트웨이) | 온라인 결제 처리 | "PG 연동"으로 언급 |
| SMS / 알림톡 | 주문/배송/장착 알림 | 명시적 언급 |
