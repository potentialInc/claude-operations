| 장바구니 관리 | 장바구니 | 수량변경/삭제 |

| 무료 장착점 찾기 | 무료장착점 | 지도+지역 검색 |
| 장착 예약 | 장착점 상세 / 주문 후 | 장착점 배송 선택 시 |
| 리뷰 작성 | 마이페이지 > 리뷰 / 구매후기 | |

## 장착점 파트너 (Partner)
| 기능 | 화면 | 비고 |
|------|------|------|
| 대시보드 | 파트너 > 대시보드 | 예약현황/정산 요약 |
| 장착 예약 확인 | 파트너 > 장착 예약 관리 | 예약 목록/상세 |
| 장착 완료 처리 | 파트너 > 장착 예약 관리 | 상태 변경 |
| 정산 확인 | 파트너 > 정산 관리 | 정산 내역/금액 |
| 매장 정보 관리 | 파트너 > 매장 정보 관리 | 영업시간/주소/연락처 |
