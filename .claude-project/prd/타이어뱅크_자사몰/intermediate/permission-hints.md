# Permission Hints — 타이어뱅크 자사몰

## 역할 목록
| 역할 | 설명 | 출처 |
|------|------|------|
| Guest | 비로그인 사용자 — 상품 조회, 장착점 조회 가능 | 화면 구조에서 추론 (Public 라우트) |
| Customer | 일반 고객 — 타이어 검색, 주문, 리뷰 작성, 장착점 조회 | 입력파일 유저타입 |
| Partner | 장착점 파트너 — 장착 예약 확인, 장착 완료 처리, 정산 확인, 매장 정보 관리 | 입력파일 유저타입 |
| Admin | 관리자 — 상품/주문/회원/장착점/정산/콘텐츠 전체 관리 | 입력파일 유저타입 |

## 명시적 권한 규칙
| 역할 | 가능 | 불가능 | 출처 |
|------|------|--------|------|
| Customer | 타이어 검색, 주문, 리뷰 작성, 장착점 조회 | 장착점 관리, 상품 관리, 정산 관리 | 입력파일 유저 설명 |
| Partner | 장착 예약 확인/완료, 정산 확인, 매장 정보 관리 | 상품 관리, 회원 관리, 다른 장착점 데이터 | 입력파일 유저 설명 |
| Admin | 전체 관리 | 없음 | 입력파일 유저 설명 |

## 소유권 규칙
| 규칙 | 출처 |
|------|------|
| 고객은 본인의 주문/리뷰/문의/차량만 관리 가능 | 마이페이지 구조에서 추론 |
| 장착점 파트너는 본인 매장의 예약/정산만 확인 가능 | 파트너 화면 구조에서 추론 |
| 관리자는 모든 데이터에 접근 가능 | 입력파일 관리자 설명 |
