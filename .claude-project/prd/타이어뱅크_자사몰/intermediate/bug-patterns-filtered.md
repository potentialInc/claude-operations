# Bug Patterns Filtered — 타이어뱅크 자사몰

> 타이어뱅크 자사몰과 관련된 카테고리의 버그 패턴만 필터링

## Category: 인증/로그인

### Pattern: 세션 만료/자격증명 오류 반복 노출
- **빈도**: 5회 (높음)
- **예방 스펙**: 세션 만료(401/403) 응답 수신 시 자동으로 토큰 갱신 시도, 갱신 실패 시 로그인 페이지로 리다이렉트, 에러 메시지 1회만 표시, 에러 상태 전역 관리

### Pattern: 로그인/회원가입 동작 불가
- **빈도**: 5회 (높음)
- **예방 스펙**: 로그인/회원가입 버튼은 폼 유효성 무관하게 클릭 가능, 검사 실패 시 각 필드에 구체적 오류 메시지 표시. 로그아웃 API는 토큰 만료 여부와 무관하게 성공 응답 반환

### Pattern: 인증 후 잘못된 화면으로 리다이렉트
- **빈도**: 4회 (중간)
- **예방 스펙**: 모든 라우트에 인증 가드 적용. 미인증→로그인 리다이렉트, 인증된 사용자→홈 리다이렉트. 회원가입 완료 후 네비게이션 스택 초기화

### Pattern: 비밀번호 유효성 검사 미흡
- **빈도**: 3회 (중간)
- **예방 스펙**: 비밀번호 입력 필드에 실시간 유효성 검사, 규칙 미충족 시 구체적 안내 메시지

## Category: 리스트/페이지네이션

### Pattern: 페이지네이션 미구현 및 정렬 오류
- **빈도**: 6회 (높음)
- **예방 스펙**: 목록 조회 API는 page, page_size 파라미터 지원, 기본 정렬 created_at DESC. 10개 이상 데이터 예상되는 모든 목록에 페이지네이션 구현

### Pattern: 필터 적용 후 자동 초기화
- **빈도**: 5회 (높음)
- **예방 스펙**: 필터 조건은 상위 컴포넌트 또는 전역 상태에 보관, 모달 재오픈 시 이전 선택값 유지. 필터 초기화 버튼 별도 제공

### Pattern: 뒤로가기 시 페이지/검색 상태 초기화
- **빈도**: 2회 (낮음)
- **예방 스펙**: 목록 페이지의 페이지 번호, 검색어, 필터 조건은 URL query parameter 또는 전역 상태에 저장하여 뒤로가기 후 복원

### Pattern: 관리자 테이블 Export 버튼 미동작
- **빈도**: 2회 (낮음)
- **예방 스펙**: 버튼에 onClick 핸들러 없으면 비활성 표시 또는 숨김. Export 기능은 CSV/Excel 다운로드 API 연결 후 활성화

## Category: 결제/인앱구매

### Pattern: 결제 완료 후 UI 미반영 및 히스토리 오류
- **빈도**: 7회 (높음)
- **예방 스펙**: 결제 완료 웹훅 수신 후 즉시 결제 상태 DB 반영, 클라이언트에 결과 전달. 결제 완료 시 성공/실패 메시지 명확히 표시, 결제 내역 페이지로 자동 이동

### Pattern: 쿠폰/프로모션 적용 오류 및 가격 계산 오류
- **빈도**: 5회 (중간)
- **예방 스펙**: 모든 금액 계산은 정수(최소 단위: 원) 기반. 결제 전 화면에서 모든 가격 항목 명시적 표시

### Pattern: 주문 상태 표시 불일치
- **빈도**: 3회 (중간)
- **예방 스펙**: 주문 상태 Enum을 단일 소스로 관리, 상태 전환 시 허용된 전환 경로만 허용하는 상태 머신 구현. 정의되지 않은 상태는 폴백 표시
