# Parsed Input — 타이어뱅크 자사몰

## Basic Info
- **앱 이름**: 타이어뱅크 자사몰
- **타입**: Web App (반응형 웹)
- **마감일**: 미정
- **유저타입**: 일반 고객, 장착점 파트너, 관리자
- **유저 관계**: 고객 ↔ 장착점 파트너 (장착 예약 기반 1:N), 관리자 → 전체 관리

## Design Reference
- **참고 사이트**: https://abctire.co.kr/
- **차별점**: 클라이언트가 명시하지 않음 (레퍼런스와 유사 구조)
- **색상/폰트**: 미정 (레퍼런스 기준: 화이트 배경, 다크그레이 텍스트, 파란색 강조)
- **레이아웃 참고**: 메인/상품목록/상세/주문결제/장착점 화면은 레퍼런스 시안과 최대한 맞춤
- **레퍼런스 사이트 구조 분석**:
  - GNB: 타이어, 무료장착점(/mount), 이벤트(/event), 고객센터(/cs), 구매후기(/reviews)
  - 메인 히어로: 슬라이더 배너 + 타이어 검색바 오버레이
  - 빠른메뉴: EASY 바로구매, 무료장착점 찾기, 타이어 장착절차, 타이어 사이즈 확인법, 타이어 자가진단
  - BEST 5 상품 섹션
  - 중간 프로모션 배너
  - 고객후기 섹션 (누적이용자/판매수/장착점 수 통계 + 후기 카드)
  - 푸터: 회사정보, 약관, SNS 링크

## Features

### 일반 고객
- 차량번호 입력 → 차량 정보 조회
- 차량에 맞는 타이어 사이즈 자동 추천
- 해당 사이즈 기준 상품 노출
- 타이어 검색 / 필터 / 정렬
- 타이어 상세페이지 (생산년도/주차, 리뷰, 성능등급)
- 장바구니 / 주문결제 / 주문완료
- 배송 선택: 자택 배송 / 장착점 배송
- 장착점 배송 시 예약/장착 연결
- 무료 장착점 찾기 (지역 검색 + 지도 기반)
- 리뷰 작성
- 이벤트 조회
- 공지사항 / FAQ / 1:1 문의
- 마이페이지: 주문내역, 차량관리, 리뷰, 문의내역

### 장착점 파트너
- 장착 예약 확인
- 장착 완료 처리
- 정산 확인
- 매장 정보 관리

### 관리자
- 대시보드
- 회원 관리
- 장착점 관리
- 상품 관리
- 주문 관리
- 정산 관리
- 이벤트 / 공지 / FAQ / 배너 / 리뷰 / 문의 관리
- CSV / 엑셀 다운로드

### 인증
- 미정 (클라이언트 미언급)

### 커뮤니케이션
- SMS / 알림톡 연동

## Data
- **수집 데이터**: 차량정보, 주문정보, 결제정보, 리뷰, 문의, 장착점정보, 장착예약, 정산
- **내보내기 데이터**: CSV / Excel (관리자)

## Technical
- **Frontend**: React
- **Backend**: NestJS
- **DB**: PostgreSQL
- **지도**: 카카오맵 API
- **차량조회**: 외부 차량조회 API 연동
- **결제**: PG 연동
- **알림**: SMS / 알림톡 연동
- **3rd Party**: 차량조회 API, 카카오맵 API, PG(결제), SMS/알림톡

### 도메인 용어
- 장착점: 타이어 장착 서비스를 제공하는 제휴 정비소
- 장착점 배송: 고객이 구매한 타이어를 장착점으로 직접 배송하는 방식
- 자택 배송: 고객 자택으로 배송하는 방식
- 생산년도/주차: 타이어 측면에 표기된 제조일 정보 (예: 2024년 32주차)
- 성능등급: 타이어의 회전저항, 젖은노면 제동력, 소음 등급

## Other
- 확장성 고려: 차량 데이터, 장착 예약, 파트너 정산, 추가 상품 카테고리, 쿠폰/포인트, 앱 전환, 디자인 QA/구현본 리뷰
