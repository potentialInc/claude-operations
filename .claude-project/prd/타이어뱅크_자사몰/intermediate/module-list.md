# Module List — 타이어뱅크 자사몰

## 시스템 모듈

| 모듈명 | 설명 | 관련 유저타입 |
|--------|------|-------------|
| 인증 모듈 | 회원가입, 로그인, 비밀번호 찾기, 토큰 관리 | All |
| 차량 조회 모듈 | 차량번호 기반 차량 정보 조회, 타이어 사이즈 매칭 | Customer |
| 상품 관리 모듈 | 타이어 상품 CRUD, 재고 관리, 카테고리/브랜드 관리 | Admin |
| 상품 검색 모듈 | 타이어 검색, 필터링, 정렬, 사이즈 기반 추천 | Customer |
| 장바구니 모듈 | 장바구니 추가/수정/삭제, 수량 관리 | Customer |
| 주문/결제 모듈 | 주문 생성, PG 결제, 배송방식 선택, 주문 상태 관리 | Customer, Admin |
| 배송 관리 모듈 | 자택 배송 / 장착점 배송 관리, 배송 상태 추적 | Customer, Admin |
| 장착점 관리 모듈 | 장착점 등록/승인, 매장 정보 관리, 지도 기반 조회 | Partner, Admin |
| 장착 예약 모듈 | 장착 예약 생성/확인/완료 처리 | Customer, Partner |
| 정산 모듈 | 장착점 정산 처리, 정산 내역 관리 | Partner, Admin |
| 리뷰 모듈 | 구매후기 작성/조회/관리, 별점 | Customer, Admin |
| 콘텐츠 관리 모듈 | 이벤트, 공지사항, FAQ, 배너 CRUD | Admin |
| 문의 모듈 | 1:1 문의 작성/답변/관리 | Customer, Admin |
| 알림 모듈 | SMS, 알림톡 발송 (주문/배송/장착 알림) | All |
| 마이페이지 모듈 | 주문내역, 차량관리, 리뷰, 문의내역, 회원정보 | Customer |

## 3rd Party API

| 서비스 | 목적 | 연동 포인트 |
|--------|------|-----------|
| 차량조회 API | 차량번호 → 차량 정보 + 타이어 사이즈 조회 | 메인 검색, 차량관리 |
| 카카오맵 API | 장착점 지도 표시, 지역 검색 | 무료장착점 찾기 |
| PG (결제 게이트웨이) | 온라인 결제 처리 | 주문/결제 |
| SMS / 알림톡 API | 주문/배송/장착 알림 발송 | 알림 모듈 |
