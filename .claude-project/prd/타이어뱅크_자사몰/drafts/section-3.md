## 3. User Application

### 3.1 Page Architecture

**Stack:** React 18, React Router v6, Zustand (전역 상태관리), TanStack Query (서버 상태), Tailwind CSS (스타일링)

#### Route Groups

| Group | Access |
|-------|--------|
| Public | 누구나 접근 가능 (비로그인 포함) |
| Auth | 비인증 사용자만 접근 가능 (로그인 시 홈으로 리다이렉트) |
| Protected(Customer) | 로그인한 일반 고객만 접근 가능 |
| Protected(Partner) | 로그인한 장착점 파트너만 접근 가능 |

#### Page Map

**Public**

| Route | Page | Access |
|-------|------|--------|
| `/` | 메인 (Home) | Anyone |

### 3.2 Feature List by Route

---

#### `/` — 메인 (Home)

**접근 권한:** Public (Anyone)

1) **히어로 배너 슬라이더**
   - **컴포넌트 상세:**
     - 풀 와이드 이미지 슬라이더 (자동 재생 5초 간격)
     - 인디케이터 도트: default(회색) / active(파란색)
     - 좌우 화살표 버튼: default(반투명 흰색) / hover(불투명 흰색) / disabled(해당 없음 — 무한 루프)
   - **데이터 표시:** 배너 이미지, 링크 URL, 순서 (관리자 설정 순서)
   - **인터랙션:** 좌우 스와이프(모바일), 화살표 클릭, 도트 클릭, 배너 클릭 시 링크 이동
   - **네비게이션:** 배너 클릭 → 설정된 URL (이벤트 상세, 상품 목록 등)

2) **타이어 검색바 오버레이**
   - **컴포넌트 상세:**
     - 차량번호 입력 필드: default(플레이스홀더 "차량번호를 입력하세요") / active(파란 테두리) / error(빨간 테두리 + 에러 메시지)
     - 검색 버튼: default(파란 배경) / hover(진한 파란) / disabled(회색, 입력 없을 때)
     - 사이즈 직접 입력 탭: 폭/편평비/인치 드롭다운 3개
   - **데이터 표시:** 차량번호 입력 결과 → 차량 정보(차종, 연식) + 호환 타이어 사이즈
   - **인터랙션:** 탭 전환(차량번호/사이즈 직접입력), 엔터키 검색, 버튼 클릭 검색
   - **네비게이션:** 검색 완료 → `/tire?size={사이즈}&vehicle={차량번호}`

3) **빠른메뉴 아이콘 그리드**
   - **컴포넌트 상세:**
     - 아이콘+텍스트 카드 5개: EASY 바로구매, 무료장착점 찾기, 타이어 장착절차, 타이어 사이즈 확인법, 타이어 자가진단
     - 카드: default(흰 배경) / hover(그림자 확대)
   - **인터랙션:** 클릭
   - **네비게이션:** 각 카드 클릭 → `/tire`, `/mount`, 가이드 모달, 가이드 모달, 자가진단 모달

4) **BEST 5 상품 섹션**
   - **컴포넌트 상세:**
     - 상품 카드 5개: 썸네일, 브랜드, 상품명, 가격, 할인율 배지
     - 카드: default / hover(그림자+약간 상승 효과)
   - **데이터 표시:** 판매량 기준 상위 5개 상품, 가격(원화 포맷)
   - **인터랙션:** 카드 클릭
   - **네비게이션:** 카드 클릭 → `/tire/:id`

5) **중간 프로모션 배너**
   - **컴포넌트 상세:** 단일 이미지 배너 (클릭 가능)
   - **인터랙션:** 클릭 → 설정된 링크 이동

6) **고객 후기 섹션**
   - **컴포넌트 상세:**
     - 통계 카운터: 누적이용자 수, 총 판매수, 장착점 수 (숫자 카운트업 애니메이션)
     - 후기 카드 슬라이더: 작성자명(마스킹), 별점, 후기 내용 미리보기
   - **데이터 표시:** 최신 후기 10건, 별점순 정렬
   - **인터랙션:** 슬라이더 좌우 스와이프/클릭
   - **네비게이션:** "더보기" 클릭 → `/reviews`, 카드 클릭 → `/reviews`

7) **GNB (글로벌 네비게이션 바)**
   - **컴포넌트 상세:**
     - 로고(홈 링크), 메뉴 항목(타이어, 무료장착점, 이벤트, 고객센터, 구매후기)
     - 로그인/회원가입 버튼 (비로그인) 또는 마이페이지/장바구니 아이콘 (로그인)
     - 장바구니 아이콘에 수량 배지: 0이면 숨김
     - 모바일: 햄버거 메뉴 → 사이드 드로어
   - **인터랙션:** 메뉴 클릭, 햄버거 메뉴 토글
   - **네비게이션:** 각 메뉴 → 해당 라우트

8) **푸터**
   - **컴포넌트 상세:** 회사정보, 이용약관/개인정보처리방침 링크, SNS 아이콘 링크
   - **인터랙션:** 링크 클릭 (새 탭)

**입력필드 검증규칙**

| Field | Type | Required | Min/Max | Pattern | Error Message |
|-------|------|----------|---------|---------|---------------|
| 차량번호 | text | ✅ | 7/9자 | 한글+숫자 조합 (예: 12가3456, 123가4567) | "올바른 차량번호 형식을 입력해주세요" |
| 타이어 폭 | select | ✅ (사이즈 직접입력 시) | — | 드롭다운 선택 | "타이어 폭을 선택해주세요" |
| 편평비 | select | ✅ (사이즈 직접입력 시) | — | 드롭다운 선택 | "편평비를 선택해주세요" |
| 인치 | select | ✅ (사이즈 직접입력 시) | — | 드롭다운 선택 | "인치를 선택해주세요" |

**상태 화면**
- **Loading:** 히어로 배너 → 회색 Skeleton 블록, BEST 5 → 카드형 Skeleton 5개, 후기 → Skeleton 카드 3개
- **Empty:** BEST 5 상품 없을 시 "인기 상품을 준비 중입니다" 메시지, 후기 0건 시 "아직 작성된 후기가 없습니다"
- **Error:** 배너/상품/후기 각 영역 독립 에러 처리 — "데이터를 불러올 수 없습니다. [다시 시도]" 버튼 표시

**에러 핸들링**
- **네트워크 에러:** 각 섹션 독립적으로 "네트워크 연결을 확인해주세요" 메시지 + 재시도 버튼
- **서버 에러:** "일시적인 오류가 발생했습니다. 잠시 후 다시 시도해주세요" 토스트 메시지
- **권한 에러:** 해당 없음 (Public 페이지)

**엣지 케이스**
- **0건:** BEST 5 섹션 숨김 처리, 후기 섹션 빈 상태 메시지 표시
- **최대치:** 배너 최대 10개 제한, 후기 슬라이더 최대 20건 로드
- **동시접근:** 읽기 전용 페이지로 동시접근 이슈 없음
- **오프라인:** 캐시된 데이터 표시, 미캐시 시 전체 오프라인 안내 화면 표시

#### `/tire` — 타이어 목록

**접근 권한:** Public (Anyone)

1) **검색/필터 영역**
   - **컴포넌트 상세:**
     - 검색바: 키워드 입력 (default: 플레이스홀더 / active: 파란 테두리 / error: 해당 없음)
     - 필터 드롭다운들: 브랜드, 사이즈, 시즌(사계절/여름/겨울), 가격대
     - 필터 초기화 버튼: default(회색 텍스트) / hover(진회색)
     - 활성 필터 태그: 선택된 필터를 태그로 표시, X 버튼으로 개별 해제
   - **인터랙션:** 드롭다운 클릭→옵션 선택, 태그 X 클릭→해제, 초기화 클릭→전체 해제, 엔터키→검색

2) **정렬 옵션**
   - **컴포넌트 상세:**
     - 정렬 드롭다운: 인기순(기본)/가격낮은순/가격높은순/최신순/리뷰많은순

   - **인터랙션:** 드롭다운 선택, 뷰 모드 토글 클릭

3) **상품 카드 그리드**
   - **컴포넌트 상세:**
     - 상품 카드: 썸네일, 브랜드 로고, 상품명, 타이어 사이즈, 가격(할인 시 원가 취소선 + 할인가), 별점, 리뷰 수
     - 카드: default / hover(그림자 확대)
     - 장바구니 담기 버튼: default(아이콘) / hover(배경색 변경)
   - **데이터 표시:** 상품 목록 (카드당: 이미지, 브랜드, 상품명, 사이즈, 가격, 별점, 리뷰수)
   - **인터랙션:** 카드 클릭→상세, 장바구니 버튼 클릭→장바구니 추가(비로그인 시 로그인 유도 모달)
   - **네비게이션:** 카드 클릭 → `/tire/:id`

4) **페이지네이션**
   - **컴포넌트 상세:**
     - 페이지 번호 버튼: default(회색) / active(파란 배경 흰 텍스트) / disabled(연회색, 첫/마지막 페이지에서 이전/다음)

#### `/tire/:id` — 타이어 상세

**접근 권한:** Public (Anyone)

1) **상품 이미지 갤러리**
   - **컴포넌트 상세:**
     - 메인 이미지: 확대 가능 (클릭 시 라이트박스)
     - 썸네일 리스트: default(테두리 없음) / active(파란 테두리)
   - **인터랙션:** 썸네일 클릭→메인 이미지 변경, 메인 이미지 클릭→라이트박스, 스와이프(모바일)

2) **상품 정보 영역**
   - **컴포넌트 상세:**
     - 브랜드명, 상품명, 타이어 사이즈 표시
     - 가격: 원가(취소선, 할인 시), 할인가, 할인율 배지
     - 성능등급 표시: 회전저항/젖은노면제동력/소음 등급 (아이콘+텍스트)
     - 생산년도/주차 표시
     - 수량 선택: - / 수량 / + 버튼, 최소 1 최대 20
   - **데이터 표시:** 브랜드, 상품명, 사이즈, 가격, 할인율, 성능등급(3개), 생산년도/주차, 재고상태

3) **구매 버튼 영역**
   - **컴포넌트 상세:**

   - **인터랙션:** 드롭다운 선택→해당 월 정산 내역 로드

3) **정산 내역 테이블**
   - **컴포넌트 상세:**
     - 테이블: 정산 기간, 장착 완료 건수, 정산 금액, 정산 상태(정산대기/정산완료/지급완료), 지급일
     - 상태 배지: 정산대기(주황)/정산완료(파란)/지급완료(초록)
   - **데이터 표시:** 정산 기간, 건수, 금액, 상태, 지급일
   - **정렬:** 정산 기간 DESC (최근 정산 우선) [💡 권장적용: 월 단위 정산]
   - **페이지네이션:** 페이지 번호 (12건/페이지)
   - **인터랙션:** 행 클릭→정산 상세 모달 (장착 완료 건 목록)

4) **정산 상세 모달**
   - **컴포넌트 상세:**
     - 정산 기간 내 장착 완료 건 목록: 예약번호, 고객명, 상품명, 장착완료일, 정산금액
     - 합계 금액
   - **인터랙션:** 스크롤, 닫기

**입력필드 검증규칙**

해당 없음 (조회 전용)

**상태 화면**
- **Loading:** 요약 카드 Skeleton + 테이블 Skeleton
- **Empty:** "정산 내역이 없습니다"
- **Error:** "정산 정보를 불러올 수 없습니다. [다시 시도]"

**에러 핸들링**
- **네트워크 에러:** 재시도 버튼
- **서버 에러:** 토스트 + 재시도
- **권한 에러:** 비파트너 접근 → 리다이렉트

**엣지 케이스**
- **0건:** 신규 장착점 (장착 완료 건 없음) → 빈 상태 메시지
- **최대치:** 페이지네이션 처리
- **동시접근:** 읽기 전용
- **오프라인:** 캐시 데이터 표시

#### `/partner/shop` — 매장 정보 관리

**접근 권한:** Protected (Partner)
**소유권 규칙:** 본인 매장 정보만 수정 가능 (`shop.partner_id === currentPartner.id`)

1) **매장 기본 정보**
   - **컴포넌트 상세:**
     - 매장명 (읽기 전용 — 관리자만 변경 가능)
     - 전화번호 입력: default / active / error
     - 주소 입력: 우편번호 검색 + 상세주소
     - 매장 소개 텍스트에어리어
   - **인터랙션:** 필드 수정, 주소 검색 모달

2) **영업시간 설정**
   - **컴포넌트 상세:**
     - 요일별 영업시간: 시작 시간 / 종료 시간 타임피커, 휴무 체크박스
     - 타임피커: 30분 단위 드롭다운
     - 휴무 체크 시 시간 선택 비활성화
   - **인터랙션:** 타임피커 선택, 휴무 체크박스 토글

3) **매장 이미지 관리**
   - **컴포넌트 상세:**
     - 이미지 업로드 영역 (드래그&드롭), 미리보기 그리드
     - 이미지: 삭제(X) 버튼, 드래그로 순서 변경
   - **인터랙션:** 파일 드래그&드롭/클릭 선택, 삭제 X 클릭, 드래그 순서 변경

4) **저장 버튼**
   - **컴포넌트 상세:** default(파란) / hover(진파란) / disabled(변경 없음) / loading(스피너)
   - **인터랙션:** 클릭→저장, 변경사항 없으면 비활성화

**입력필드 검증규칙**

| Field | Type | Required | Min/Max | Pattern | Error Message |
|-------|------|----------|---------|---------|---------------|
| 전화번호 | tel | ✅ | 9/11자 | 숫자+하이픈 | "올바른 전화번호를 입력해주세요" |
| 주소 | text | ✅ | — | — | 다음 주소 API 자동 입력 |
| 상세주소 | text | ✅ | 1/100자 | — | "상세주소를 입력해주세요" |
| 매장 소개 | textarea | ❌ | 0/1000자 | — | — |
| 영업 시작시간 | time | ✅ (비휴무일) | — | HH:mm | "영업 시작 시간을 선택해주세요" |
| 영업 종료시간 | time | ✅ (비휴무일) | — | HH:mm, 시작보다 늦어야 함 | "종료 시간은 시작 시간 이후여야 합니다" |

**상태 화면**
- **Loading:** 전체 폼 Skeleton
- **Empty:** 해당 없음 (기존 매장 정보 항상 존재)
- **Error:** "매장 정보를 불러올 수 없습니다. [다시 시도]"

**에러 핸들링**
- **네트워크 에러:** 저장 실패 시 토스트 "네트워크 연결을 확인해주세요", 입력 데이터 유지
- **서버 에러:** "매장 정보 저장에 실패했습니다. 다시 시도해주세요", 입력 데이터 유지
- **권한 에러:** 비파트너 접근 → 리다이렉트

**엣지 케이스**
- **0건:** 해당 없음
- **최대치:** 이미지 10장 제한, 파일 크기 5MB 제한
- **동시접근:** 여러 파트너(같은 매장)가 동시 수정 시 최종 저장 우선 + "정보가 업데이트되었습니다. 다시 확인해주세요" [💡 권장적용]
- **오프라인:** 폼 표시(읽기), 저장 불가 + "오프라인 상태에서는 저장할 수 없습니다"
